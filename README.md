# 🎬 CineMatch — Movie Recommendation System

A beginner-to-intermediate level **Content-Based Movie Recommendation System**
built with Python, Scikit-learn and Streamlit. This project was built as a
learning/internship-style project to demonstrate a complete, real-world
machine learning workflow — from raw data to a deployed web app.

---

## 📌 Project Overview

CineMatch recommends movies similar to one you already like. It does this
by analyzing each movie's **genres, keywords, overview, cast and director**,
converting that combined text into numeric vectors using **TF-IDF**, and
then measuring how similar movies are to each other using **Cosine
Similarity**.

Type or select a movie you enjoyed, and CineMatch will suggest the
**Top 10 most similar movies** from the dataset — even if you make a small
spelling mistake while typing!

---

## ✨ Features

- **Content-Based Filtering** using genres, keywords, overview, cast & director
- **TF-IDF Vectorization** to turn text into meaningful numeric features
- **Cosine Similarity** to measure how alike two movies are
- **Fuzzy / typo-tolerant search** using Python's `difflib`
- Clean, modular code split across multiple files (not a single messy script)
- Interactive **Streamlit web app** with:
  - Searchable movie dropdown (type-to-filter suggestions)
  - Manual text input for typing movie names directly
  - Top 10 recommendations shown as expandable cards
  - "🎲 Surprise Me" random movie recommendation button
  - Recently searched movies list
  - Recommendation history for the current session
  - Dataset statistics in the sidebar
  - Reset button to clear session history
  - Friendly success/error messages and loading spinners
- Proper exception handling throughout
- Preprocessed data & similarity matrix saved with **Pickle** for fast app startup

## 🎞 Dataset Information

This project uses a **TMDB-style movie metadata dataset** containing 2,000
popular movies, with the following key columns:

| Column          | Description                                   |
|-----------------|------------------------------------------------|
| `id`            | Unique movie ID                                |
| `title`         | Movie title                                    |
| `genres`        | Genre(s) of the movie                          |
| `keywords`      | Plot-related keywords                          |
| `overview`      | Short plot summary                             |
| `cast`          | Leading cast members                           |
| `director`      | Movie director                                 |
| `popularity`    | Popularity score                               |
| `vote_average`  | Average user rating (out of 10)                |
| `release_date`  | Release date                                   |

The raw data intentionally contains some missing values (e.g. missing
`keywords` or `director`), which are handled during preprocessing — this
mirrors what you'd encounter with a real-world dataset such as the public
[TMDB 5000 Movies Dataset](https://www.themoviedb.org/).

The dataset file is located at `dataset/movies.csv`.

   ```bash

   source venv/bin/activate      # On Windows: venv\Scripts\activate
   ```

   ```bash

   ```

---

## ▶️ Running Instructions

**Step 1 — Preprocess the data (only needs to be run once, or whenever the dataset changes):**

```bash
python preprocess.py
```

This will clean the dataset, compute the TF-IDF + cosine similarity matrix,
and save the results into the `models/` folder.

> Note: This repository already ships with pre-generated `models/*.pkl`
> files, so this step is optional unless you modify `dataset/movies.csv`.

**Step 2 — Launch the Streamlit app:**

```bash
streamlit run app.py
```

Then open the URL shown in your terminal (usually `http://localhost:8501`)
in your web browser.
